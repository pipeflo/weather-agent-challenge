# Weather Agent Scenario Workshop

**Workshop Type**: Realistic Development Scenario  
**Duration**: 60 minutes  
**Focus**: Inheriting and completing incomplete codebases with Claude Code

## Workshop Structure

### For Participants
- **Start here**: `cd weather-agent/` - Contains the inherited codebase
- **Follow**: `PARTICIPANT_GUIDE.md` - Your workshop instructions

### For Instructors
- **Teaching Guide**: `INSTRUCTOR_GUIDE.md` - Workshop flow and concepts
- **Private Guide**: `INSTRUCTOR_PRIVATE_GUIDE.md` - Exact prompts and solutions
- **Solution**: `SOLUTION.md` - Complete implementation reference

## Scenario

You're a new developer who inherited an incomplete weather agent from a colleague who left the company. Your task is to complete the implementation and deploy it to production using Claude Code as your development partner.

## Quick Start

**Participants**:
```bash
git clone https://github.com/pipeflo/weather-agent-challenge.git
cd weather-agent-challenge/weather-agent
claude
```

**Instructors**: Review the guide files in the root directory before starting the workshop.
